# MAGIC MIRROR PROJECT

SETUP INSTRUCTIONS:

1. Install Dependencies:
   Open a terminal in this folder and run:
   pip install -r requirements.txt

2. Setup API Key:
   - Rename the file '.env.example' to '.env'
   - Open it and paste your WeatherAPI key inside.

3. Setup Faces:
   - Go to the 'known_faces' folder.
   - Add a photo of yourself and name it 'YOURNAME.jpg' (e.g., TEACHER.jpg).

4. Run the Project:
   - To start the mirror: python main.py
   - To send commands: python admin.py
